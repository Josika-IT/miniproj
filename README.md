##ATM Management System 
Team Details
Team Number: TEAM 5
Project Title: ATM Management System
----------
Objective
To design and implement a menu-driven ATM management application in C that performs basic banking operations using structures and file handling.
-----------
## Features
- Create a new bank account
- Deposit amount
- Withdraw amount
- Balance enquiry
- File-based data storage
----------
## Technologies Used
- C Programming Language
- File Handling
- Structures
----------
## Input
- Account Number
- Account Holder Name
- Amount for deposit
- Amount for withdrawal
--------
## Output
- Account creation confirmation
- Deposit and withdrawal status messages
- Account balance display
- Error messages for invalid operations
-------
## How to Run
1. Compile the program using a C compiler  
2. Run the executable  
3. Follow the on-screen ATM menu
--------
